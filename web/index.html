API 请求文档

兼容 OpenAI / Anthropic API 格式，任何支持自定义 Base URL 的客户端均可直接使用。

基本信息

Base URL{your-domain}（部署后替换为实际地址）
认证方式Authorization: Bearer {any-key}（无需真实 Key，随便填）
Content-Typeapplication/json

端点一览

方法 路径 说明 兼容格式
GET /v1/models 获取支持的模型列表 OpenAI
POST /v1/chat/completions 文本对话（支持流式） OpenAI
POST /v1/messages 文本对话（Claude 格式） Anthropic
POST /v1/images/generations 图片生成 OpenAI
POST /v1/videos/generations 视频生成 自定义

1. 文本对话 POST /v1/chat/completions

兼容 OpenAI Chat Completions API，支持流式和非流式输出。

请求参数

参数类型必填说明
modelstring是模型 ID，如 gpt-4o-mini、claude-sonnet-4-20250514
messagesarray是对话消息数组，每项含 role 和 content
streamboolean否是否流式输出，默认 false
temperaturenumber否温度参数，0~2
max_tokensinteger否最大输出 token 数
请求示例 
POST /v1/chat/completions
Content-Type: application/json
Authorization: Bearer any-key

{
  "model": "gpt-4o-mini",
  "messages": [
    { "role": "system", "content": "You are a helpful assistant." },
    { "role": "user", "content": "你好，请自我介绍" }
  ],
  "stream": false,
  "temperature": 0.7
}
响应示例 
{
  "id": "chatcmpl-xxx",
  "object": "chat.completion",
  "model": "gpt-4o-mini",
  "choices": [
    {
      "index": 0,
      "message": {
        "role": "assistant",
        "content": "你好！我是一个 AI 助手..."
      },
      "finish_reason": "stop"
    }
  ],
  "usage": {
    "prompt_tokens": 20,
    "completion_tokens": 50,
    "total_tokens": 70
  }
}
cURL 示例 
curl -X POST https://your-domain/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer any-key" \
  -d '{
    "model": "gpt-4o-mini",
    "messages": [{"role": "user", "content": "hello"}],
    "stream": false
  }'

2. Claude 格式对话 POST /v1/messages

兼容 Anthropic Messages API 格式，支持 SSE 流式输出。

请求参数

参数类型必填说明
modelstring否模型 ID，默认 claude-opus-4-5-20251001
messagesarray是对话消息数组
streamboolean否是否流式输出
max_tokensinteger否最大输出 token 数，默认 8192
systemstring否系统提示词
请求示例 
POST /v1/messages
Content-Type: application/json
x-api-key: any-key
anthropic-version: 2023-06-01

{
  "model": "claude-sonnet-4-20250514",
  "messages": [
    { "role": "user", "content": "解释量子计算" }
  ],
  "max_tokens": 4096,
  "stream": true
}
cURL 示例 
curl -X POST https://your-domain/v1/messages \
  -H "Content-Type: application/json" \
  -H "x-api-key: any-key" \
  -H "anthropic-version: 2023-06-01" \
  -d '{
    "model": "claude-sonnet-4-20250514",
    "messages": [{"role": "user", "content": "hello"}],
    "max_tokens": 4096,
    "stream": true
  }'

3. 获取模型列表 GET /v1/models

返回所有支持的模型，兼容 OpenAI List Models API。

cURL 示例 
curl https://your-domain/v1/models
响应示例 
{
  "object": "list",
  "data": [
    { "id": "gpt-4o-mini", "object": "model", "owned_by": "openai" },
    { "id": "claude-sonnet-4-20250514", "object": "model", "owned_by": "anthropic" },
    ...
  ]
}

4. 图片生成 POST /v1/images/generations

兼容 OpenAI Images API，支持 DALL-E、FLUX、Stable Diffusion 等模型。

请求参数

参数类型必填说明
modelstring否模型 ID，默认 dall-e-3
promptstring是图片描述文本
ninteger否生成数量，默认 1
sizestring否图片尺寸，如 1024x1024
请求示例 
POST /v1/images/generations
Content-Type: application/json
Authorization: Bearer any-key

{
  "model": "dall-e-3",
  "prompt": "A cute cat sitting on a windowsill at sunset",
  "n": 1,
  "size": "1024x1024"
}
响应示例 
{
  "created": 1708000000,
  "data": [
    {
      "url": "https://..."
    }
  ]
}
cURL 示例 
curl -X POST https://your-domain/v1/images/generations \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer any-key" \
  -d '{
    "model": "dall-e-3",
    "prompt": "a cute cat",
    "n": 1
  }'

5. 视频生成 POST /v1/videos/generations

通过 Together AI 驱动，支持 Sora、Veo、Kling、Hailuo 等视频模型。视频生成耗时较长（1~5 分钟），请设置足够超时时间。

请求参数

参数类型必填说明
modelstring是视频模型 ID，如 togetherai:openai/sora-2
promptstring是视频描述文本
widthinteger否视频宽度，如 1920
heightinteger否视频高度，如 1080
fpsinteger否帧率，如 24
durationnumber否时长（秒）
请求示例 
POST /v1/videos/generations
Content-Type: application/json
Authorization: Bearer any-key

{
  "model": "togetherai:minimax/hailuo-02",
  "prompt": "A cinematic drone shot of a futuristic city at night",
  "width": 1920,
  "height": 1080,
  "fps": 24
}
响应示例 
{
  "created": 1708000000,
  "data": [
    {
      "url": "https://..."
    }
  ]
}
cURL 示例 
curl -X POST https://your-domain/v1/videos/generations \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer any-key" \
  -d '{
    "model": "togetherai:minimax/hailuo-02",
    "prompt": "a drone shot of a city",
    "width": 1920,
    "height": 1080
  }'

6. 客户端接入指南

客户端配置方式
ChatBox设置 > API Provider > OpenAI API Compatible > API Host 填部署地址
NextChat设置 > 接口地址填部署地址，API Key 随便填
LobeChat设置 > 语言模型 > OpenAI > API Proxy 填 {domain}/v1
Cursor / Windsurf设置中 OpenAI Base URL 填 {domain}/v1
one-api / new-api添加渠道时 Base URL 填部署地址
Python openai 库client = OpenAI(base_url="{domain}/v1", api_key="any")

7. 注意事项

  • 每个 Puter 账号每日约有 1000 次请求限制，高级模型（opus/pro/o1）消耗更快
  • 推荐使用 nano/mini/flash/haiku 级别模型，额度更宽裕
  • 图片生成约 5 张/天，视频生成额度更少，请省着用
  • 视频生成耗时 1~5 分钟，请设置足够的超时时间（建议 300 秒）
  • 多账号轮询可线性扩展总额度：10 个账号 = 10,000 次/天
  • Authorization 头无需真实 API Key，系统使用管理面板中配置的 Puter Token

添加账号

支持直接粘贴 JWT Token，或从浏览器复制的 curl 命令（会自动提取 puter_auth_token）
